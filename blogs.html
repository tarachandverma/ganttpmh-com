(1) Project Plan let project manager visualize road map for various milestones. From the vast ocean of Project Management, we have selected 10 best steps to Plan a Project:-

(2) Raw project milestones shall be jotted down. Discuss with key stakeholders about their feasibility and importance

(3) Read the scope of project and BOQ (if available), chunked milestones into WBS and activities

(4) Make logical sequence, provide duration based on quantities to be executed for activity

(5) Feed roles and resources required for all individual activities. Check this with Project Manager and stakeholders. Mostly changes would be considered in duration

(6) Freeze the the baseline, take print out show to client and proceed for monitoring progress

(7) There would schedule, cost baseline, variance plan, earned value analysis to be reported.

(8) Check the project quality and risks through the assessment. Prepare communication channel to be followed for risk mitigation

(9) Feed Progress as per the resourced consumed or task percent completed into the plan. Prepare S curve and present before key stakeholder

(10) Generate report of delay, catch plan, weekly forcasted schedule, S curve etc. If communication is maintained through proper reporting, there wont be any delay in Project.
